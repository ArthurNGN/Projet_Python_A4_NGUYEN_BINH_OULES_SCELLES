PROJET PYTHON FOR DATA ANALYSIS ESILV A4 2022

======================================================

DATASET: ONLINE NEWS POPULARITY
lien: https://archive.ics.uci.edu/ml/datasets/Online+News+Popularity

Ce dataset résume un ensemble hétérogène d'articles publiés par Mashable sur une période de deux ans.
On y trouve de nombreux paramètres qui traitent aussi bien du fond, de la forme comme des infos de publication de l'article : Comme le jour de la semaine où l'article a été publié, jusqu'à la longueur moyenne des mots contenus dans l'article.

======================================================

But du projet:
-Produire des visuels des données
-Tenter plusieurs solutions de machine learning pour prédire le succès d'un futur article
-Transformer le modèle en une API (via Django ou Flask)

======================================================

Visuels:

Une dizaine de visuels ont été produits, ils nous permettent chacun de tirer des conclusions sur les aspects du dataset qui nous semble les plus probant comme notamment la longueur de l'article idéal.


Prédiction/Modeling:

Nous avons tenté de produire les modèles de machine learning suivants : Random Forest, Decision Tree, KNN, Logistic Regression, Perceptron, Gaussienne, SGD, SVC
Les meilleures modélisations que nous avons obtenu reposent sur les modèles de random forest et de decision tree.


API:

Nous avons fait le choix de ne pas traiter cette partie du travail.
